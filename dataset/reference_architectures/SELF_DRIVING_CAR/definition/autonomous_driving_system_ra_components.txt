ML, AI, DL Algorithms: Provide intelligent perception, prediction, and decision-making capabilities. They process collected data to enable automation and adaptive behavior.

Data Collection: Gathers data from sensors, internal networks, and external sources. It prepares structured inputs for analytics and control functions.

UI/UX and Infotainment: Delivers user interaction and multimedia functionalities. It enables visualization, configuration, and user engagement with the system.

Real-time and Critical Control Software: Executes time-sensitive and safety-critical functions. It ensures deterministic behavior and reliable system operation.

Software Frameworks and Standards (AUTOSAR, ROS, ROS2, RTOS, etc.): Provide middleware and standardized interfaces for application development. They manage communication, scheduling, and hardware abstraction.

Internal Networking Interfaces (CAN, LIN, GigaEthernet, USB 3.X, etc.): Enable data exchange between hardware and software components. They support heterogeneous communication protocols and bandwidth requirements.

Sensors: Capture physical signals from the environment. They provide raw data for perception, control, and analytics.

Processing Units: Execute software algorithms and control logic. They provide computational resources for real-time and AI workloads.

V2X/Cloud Communications: Enable communication with external vehicles, infrastructure, and cloud services. They support data sharing, remote updates, and cooperative functions.

Mobile Platform/Actuators: Execute physical actions based on control commands. They include propulsion, steering, braking, and other motion-related mechanisms.
